# SVG图标目录

此目录用于存放SVG图标文件，这些图标将被自动导入并注册为全局组件。

## 使用方法

1. 将SVG图标文件放置在此目录下
2. 在组件中使用 `<svg-icon icon-class="图标名称" />` 来引用图标
